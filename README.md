## üíª Development

To run this project locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/nschmed14/music-website.git
    cd music-website
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Set up environment variables:
    *   Copy the example file: `cp .env.example .env.local`
    *   Obtain your own API keys for [EmailJS](https://www.emailjs.com/) and [Google reCAPTCHA](https://www.google.com/recaptcha/admin/) and add them to the `.env.local` file.
4.  Start the development server:
    ```bash
    npm start
    ```
